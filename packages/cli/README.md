## Quick Start

Packasso config is one or set of modules.

Add `packasso` field in `package.json`:

```json
{
  "name": "pablo-picasso",
  "version": "1972.07.03",
  "packasso": "@packasso/preset-ts-tsc-uvu"
}
```

Run packasso.

```shell
npx @packasso/cli
```

Enjoy your code!
